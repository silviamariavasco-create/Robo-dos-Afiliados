# Robo dos Afiliados
**Repositório oficial do Robô Automático — o local de instalação legítimo do robô que automatiza a criação de links de afiliado no Mercado Livre.:**


🤖 Robô Automático de Afiliados – Promobit + Mercado Livre + WhatsApp
Robô de automação que busca ofertas no Promobit, gera links afiliados do Mercado Livre, cria descrições com IA (Groq) e envia mensagens para WhatsApp (contato, grupo e status) — tudo de forma automática ou semi-automática.

🚀 Visão Geral
Este projeto automatiza o processo de divulgação de ofertas como afiliado:

Busca ofertas ativas no Promobit (agregador de promoções).

Extrai o link do produto no Mercado Livre (navegando pela página do Promobit).

Gera link de afiliado usando o Link Builder do Mercado Livre (via navegador Edge persistente).

Cria uma descrição atrativa para WhatsApp com a ajuda da API Groq (LLaMA 3.3‑70B).

Envia a mensagem para um contato, grupo e/ou status do WhatsApp Web.
